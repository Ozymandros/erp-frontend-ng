## Getting Started
- Download: [antigravity.google/download](https://antigravity.google/download)
- Supported platforms:
  - **macOS**: Monterey 12+ (Apple silicon only)
  - **Windows**: Windows 10 (64‑bit)
  - **Linux**: glibc >= 2.28, libstdc++ >= 3.4.25 (Ubuntu 20, Debian 10, Fedora 36, RHEL 8)

## Basic Navigation
- Open Agent Manager from the Editor via the top‑bar button or <kbd>Cmd + E</kbd>.
- Use “Focus Editor” for a workspace, or open the editor via “Open Editor” or <kbd>Cmd + E</kbd>.
